## 1. Core Football Terminology

### The Game

| Italian | English | Usage Context |
|---------|---------|---------------|
| il calcio | football / soccer | The sport itself. Literally "kick." Derives from *calciare* (to kick) and the Renaissance game *calcio fiorentino*. |
| la partita | the match / the game | General term for any match. |
| la gara | the match / the contest | Slightly more formal; often used in written commentary. |
| la sfida | the challenge / the clash | Emphasizes competitive nature; "la sfida di stasera" = tonight's clash. |
| l'incontro | the encounter / the match | Formal register; common in print journalism. |
| il derby | the derby | Same as English; used for local rivalries. |
| l'amichevole | friendly match | Pre-season or non-competitive fixture. |

### The Ball and Goal Structure

| Italian | English | Usage Context |
|---------|---------|---------------|
| il pallone / la palla | the ball | *Pallone* is more specific to football; *palla* is generic. |
| la porta | the goal (structure) | Literally "door." The physical goal frame. |
| la rete | the net / a goal | Dual meaning: the net itself, or a goal scored ("ha segnato una rete"). |
| il palo | the goalpost | "Palo!" is shouted when a shot hits the post. |
| la traversa | the crossbar | "Traversa!" is shouted when a shot hits the bar. |
| il primo palo | the near post | The post closest to the ball. |
| il secondo palo | the far post | The post furthest from the ball. |
| il sette | the top corner | Literally "the seven" -- the angle between post and crossbar resembles the number 7. |
| il dischetto | the penalty spot | Literally "little disc." |

### The Pitch

| Italian | English | Usage Context |
|---------|---------|---------------|
| il campo | the pitch / the field | General term for the playing surface. |
| il centrocampo | the midfield / center of the pitch | Both the area and the department. |
| la fascia | the wing / the flank | Also means the captain's armband (*fascia di capitano*). |
| l'area di rigore | the penalty area / the box | The 18-yard box. |
| l'area di porta / l'area piccola | the goal area / the six-yard box | The small box in front of goal. |
| la linea di porta | the goal line | |
| la linea laterale | the touchline / sideline | |
| la bandierina d'angolo | the corner flag | |
| la lunetta | the penalty arc | The semi-circle at the edge of the penalty area. |
| la metà campo | the half / one side of the pitch | |

## 3. Tactical Terms

### Systems and Formations

| Italian | English | Usage Context |
|---------|---------|---------------|
| il modulo | the formation / the system | "Il modulo 4-3-3" = the 4-3-3 formation. |
| lo schema | the set play / the tactical scheme | Can refer to a rehearsed set-piece routine. |
| il catenaccio | the door bolt (ultra-defensive system) | Literally "door bolt." The iconic Italian defensive system popularised by Helenio Herrera's Inter Milan in the 1960s. Emphasises a sweeper (*libero*) behind the defensive line. |
| la zona mista | mixed zone defence | Blends zonal marking with man-to-man marking. Evolved from catenaccio in the 1970s. Dominated Italian football through the 1982 World Cup. |
| la zona | zonal marking / zonal defence | Defending spaces rather than tracking specific opponents. |
| la marcatura a uomo | man-to-man marking | Each defender is assigned a specific opponent. |
| la marcatura a zona | zonal marking | Defending areas of the pitch rather than specific players. |

### Tactical Concepts

| Italian | English | Usage Context |
|---------|---------|---------------|
| il contropiede | the counter-attack | One of the most important tactical terms in Italian football. |
| il pressing | pressing / high press | Borrowed from English but naturalised in Italian usage. |
| il possesso palla | ball possession | |
| il gioco corto | short passing game | |
| il gioco lungo | long-ball game | |
| la ripartenza | the quick transition / the break | A fast transition from defence to attack. |
| il palleggio | ball circulation / passing sequences | In a tactical context, refers to keeping possession through passing (different from its other meaning of juggling). |
| il fraseggio | intricate passing / short interplay | Quick, incisive short passing between players. |
| la costruzione dal basso | building from the back | Playing out from the goalkeeper and defence. |
| il lancio | the long pass / long ball | A long kick upfield. |
| la sciabolata | the long diagonal pass | Literally "sabre cut." A sweeping long pass, often from defence to attack. |
| la verticalizzazione | the vertical pass / the through ball | A pass that breaks lines and goes forward vertically through the pitch. |
| il cambio gioco | the switch of play | Switching the ball from one side of the pitch to the other. |
| la sovrapposizione | the overlap | A full-back or midfielder running past a teammate on the outside. |
| il taglio | the cutting run | A diagonal run across the defence to receive a pass. |
| lo smarcamento | the movement off the ball / losing your marker | The act of getting free from a defender. |
| la diagonale | the diagonal run / covering run | A defender's diagonal covering movement. |
| la copertura | defensive cover | |
| il raddoppio | the double team | Two players pressing the ball carrier. |

| la fase difensiva | the defensive phase | |

## 12. Famous Commentator Signatures

Italian football commentary has a distinctive emotional style. Recognising these signature phrases helps the AI system attribute and contextualise the source material.

### Fabio Caressa (Sky Sport Italia)

| Phrase | Meaning | Context |
|--------|---------|---------|
| "[Cognome] [Nome]! Gol! Gol! Gol!" | Full name + extended goal call | Caressa always says the player's surname then first name when a goal is scored, followed by repeated "Gol!" |
| "Il cielo e' azzurro sopra Berlino!" | "The sky is blue over Berlin!" | His legendary call during Italy's 2006 World Cup final victory in Berlin. |
| "Andiamo a Berlino, Beppe!" | "We're going to Berlin, Beppe!" | Said to co-commentator Beppe Bergomi after Italy's semi-final win in 2006. |
| "Campioni del mondo! Campioni del mondo! Campioni del mondo!" | "World Champions!" (x3) | His call at the final whistle of the 2006 World Cup final. |

### Sandro Piccinini (Mediaset)

| Phrase | Meaning | Context |
|--------|---------|---------|
| "Sciabolata morbida!" | "A soft sabre cut!" | His signature description of an elegant long diagonal pass. |
| "Proprio lui!" | "Him of all people!" | When an unlikely player scores or makes a decisive play. |
| "'Ccezionale!" | "Exceptional!" (contracted) | His distinctive abbreviation of *eccezionale*. |
| "Non va!" | "It won't work!" / "No good!" | Indicating an unsuccessful play. |
| "L'attesa e' finita, si comincia..." | "The wait is over, we begin..." | His classic match introduction. |

### Common Commentator Conventions

| Convention | Meaning |
|------------|---------|
| Repeating "Gol!" multiple times | The more repetitions, the more spectacular the goal. A routine tap-in might get two; a screamer gets five or more. |
| Naming the scorer with surname first | Standard Italian commentary practice: "Totti, Francesco!" |
| "Mister [surname]" for the coach | "Mister Allegri" = Coach Allegri. Always "Mister," never "Coach." |
| Using team colour nicknames | Commentators freely alternate between club name and colour nickname within the same sentence. |
| Diminutives and augmentatives | Adding suffixes for emphasis: *-ino* (small/cute), *-one* (big/great), *-accio* (ugly/spectacular depending on context). E.g., *golaccio* = a stunning goal (augmentative used positively). |
